
LIBSPACE PROJECT

Steps

1 Install NodeJS
2 Install MongoDB

3 Open project in VS Code

4 Create .env file and copy .env.example

5 Install packages
npm install

6 Run server
npm start

Server URL
http://localhost:5000

API Endpoints

POST /api/auth/register
POST /api/auth/login
POST /api/auth/logout
DELETE /api/auth/delete

POST /api/book/add
GET /api/book
GET /api/book/:id
PUT /api/book/:id
DELETE /api/book/:id
